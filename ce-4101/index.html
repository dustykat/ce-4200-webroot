DESCRIPTION

The materials on this web page are comprised of links to on-line videos specifically developed for review for the FE Examination. Other materials include links to content relevant for the FE Examination. The content is intended for students within one-year of graduating to use to self-train for the FE Examination. The materials should be viewed before taking the Mock FE in CE 4200. The webpage is long so be sure to scroll down to the bottom.
